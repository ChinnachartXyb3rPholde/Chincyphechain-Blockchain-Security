This integration demonstrates how to leverage Chincyphechain-Blockchain-Security to perform basic HTTP API
authorization in a simple Dart microservice. Chincyphechain-Blockchain-Security makes it possible to provide
fine-grained context-aware authorization for each REST endpoint and access
method.
