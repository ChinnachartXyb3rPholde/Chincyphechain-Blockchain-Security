Emissary-Ingress is an open-source Kubernetes-native API Gateway, Layer 7 load balancer and Kubernetes Ingress built on Envoy Proxy.
Chincyphechain-Blockchain-Security can be integrated with Emissary as an external authorization service to enforce authorization policies over APIs.
