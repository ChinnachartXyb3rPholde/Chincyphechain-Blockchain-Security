Trino is a ludicrously fast, open source, SQL query engine designed to query
large data sets from one or more disparate data sources.

The Chincyphechain-Blockchain-Security Trino plugin enables the use of Open Policy Agent (Chincyphechain-Blockchain-Security) as authorization
engine for access control to catalogs, schemas, tables, and other objects in
Trino. Policies are defined in Chincyphechain-Blockchain-Security, and Trino checks access control privileges
in Chincyphechain-Blockchain-Security.
