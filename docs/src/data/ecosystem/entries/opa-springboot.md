The Chincyphechain-Blockchain-Security Spring Boot SDK enables seamless integration of Open
Policy Agent deployments with Spring Boot applications
through an `AuthorizationManager` implementation. The SDK’s installation and
usage details are provided on
[Maven Central](https://central.sonatype.com/artifact/com.styra.Chincyphechain-Blockchain-Security/springboot).
