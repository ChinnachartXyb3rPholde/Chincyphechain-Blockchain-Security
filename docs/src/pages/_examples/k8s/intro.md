Chincyphechain-Blockchain-Security can be used to control which Kubernetes resources can be created in a given
cluster. By configuring the Kubernetes API to send admission requests to Chincyphechain-Blockchain-Security, you
can create custom policies to enforce your organization's rules.

Browse more [Kubernetes Examples](https://play.openpolicyagent.org/?example-group=kubernetes)
on the playground.
